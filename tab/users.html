Users Help Page

How do I add a new user?

By clicking on the “New User” button on the top-right. You need to fill in the information and choose which user-type the new user is. You can do this by clicking on the “User Type”-bar and a dropdown menu will appear. Click “Add User” to finish the process.

How do I show or edit a user?

You can double click on the user you want to show/edit or you can mark the user and click the “Show/Edit User” button on the top-right. Here you can change the details you want about a user and finish the process by clicking “Edit User”. If you do not want any changes, you can click “Cancel”.

How do I delete users?

You mark the user you want to delete and click “Delete User”.

How do I reactivate old users?

Go to the list below and find the user you want to reactivate. Mark this user and click “Reactivate User” on the right side of the window.

How do I search for users?

Write what you want to find in the “Search Users”-bar to search through the user-database.

Which user types have access to the users panel?

Only admin users have access to this panel.
